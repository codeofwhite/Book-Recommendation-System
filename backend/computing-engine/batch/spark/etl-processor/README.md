## Spark 准实时 ETL 服务：异构数据同步中心

该服务基于 **Structured Streaming** 实现准实时（Near Real-Time, NRT）的抽取、转换和加载（ETL）流程，主要目标是将数据同步到离线分析数据库中。

### 核心职责

该组件在数据架构中扮演**异构数据库同步中心**的角色，主要负责以下任务：

* **准实时数据同步：** 负责将来自多个核心业务数据库（如 **MySQL** 和 **MongoDB**）的数据进行准实时同步。
    * **覆盖数据源：** 包括但不限于用户认证（`auth`）、图书信息（`book`）以及各类用户参与行为数据（如点赞、收藏等）。
* **Debezium CDC 集成：** 集成并处理 **Debezium** 捕获的变更数据（Change Data Capture, CDC）流，确保数据同步的高效性和准确性。
