## Flask 后端服务部署与架构概览

本项目后端服务采用 **Flask 框架**进行构建和部署。

### 1. 架构与部署环境

* **核心框架：** 使用 **Python Flask** 作为后端 Web 服务的核心框架。
* **组件管理：** 整个应用生态系统和所有依赖组件（包括数据库、消息队列等）均通过 **Docker Desktop** 进行容器化创建、配置和管理。

### 2. 文档与结构指引

我们遵循清晰的文档优先原则，项目的详细信息和使用指南分散在各个模块中：

* **总览文档：** 完整的部署和架构细节请参阅主 `docs` 目录下的相关文档。
* **模块详情：** 每个主要的子功能或服务文件夹内部都包含独立的 **`README.md`** 文件，提供了针对该模块的详细说明、配置和使用方法。
