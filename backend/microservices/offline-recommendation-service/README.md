## 离线推荐引擎：核心数据分析与模型训练

该模块是推荐系统的核心智能层，负责执行复杂的批量数据分析和模型训练。

### 核心算法与策略

* **算法基础：** 采用**协同过滤 (Collaborative Filtering)** 作为核心离线推荐算法，用于挖掘用户之间的相似性或物品之间的关联性。

### 数据集成与应用

离线推荐引擎扮演着**统一数据分析平台**的角色，其主要流程如下：

1.  **数据集成 (Data Aggregation):** 接收来自各个微服务（`auth`, `book`, `user-engagement` 等）**同步汇聚**而来的全量历史数据。这些数据通常存储在数据仓库（如 ClickHouse/HDFS）中。
2.  **数据分析与应用：** 对集成的历史数据进行深度分析，并将结果应用于两个主要方向：
    * **实时/准实时推荐数据供给：** 为实时推荐系统提供训练好的模型参数、预计算的相似度数据或用户长期画像特征。
    * **业务洞察与可视化：** 对聚合数据进行分析和统计，提供支持**商业智能 (BI)** 和**数据可视化服务**所需的数据源。
