## 实时推荐 API 微服务 (Real-Time Recommendation API Service)

该微服务是一个面向前端的接口层，专注于提供低延迟、高新鲜度的推荐结果。它本身不进行复杂的计算，而是作为连接前端和实时数据存储的桥梁。

### 核心功能与架构定位

* **接口提供者：** 提供统一的 HTTP/API 接口，供前端服务调用以获取实时推荐列表。
* **数据获取：** 从 **Redis** 缓存中检索预先计算好的推荐结果。
    * **数据源：** Redis 中的数据由上游的 **Flink Streaming Job** 实时计算并写入。
    * **数据特征：** 这些数据通常包含基于用户近期行为和短期兴趣的**最新推荐资源**。
* **角色定位：** 在整体架构中，它扮演着**资源服务接口**的角色，将复杂的实时数据流处理与前端展示层解耦。
