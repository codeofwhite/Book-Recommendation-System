## 用户参与/行为记录微服务 (User Engagement Service)

该微服务专注于捕捉、记录和管理用户的各种互动行为，是用户行为数据采集的核心模块。

### 核心功能

  * **行为数据管理：** 负责处理和持久化用户的**点赞 (Like)**、**收藏 (Favorite)** 和**评论 (Review/Comment)** 等关键互动记录。
  * **数据源提供：** 为数据分析、推荐系统和其他服务提供准确、实时的用户行为数据源。

### 数据库初始化指南

由于自动建表可能存在问题，首次启动此服务后，必须执行以下数据库初始化操作以创建必要的表结构：

#### 1\. 推荐方式：使用 Flask 命令自动初始化

**推荐使用**此方法，它通过服务的内置命令自动执行建表操作：

```bash
docker exec user_engagement_service_app flask init-db
```

#### 2\. 手动 SQL 建表（备用）

如果自动初始化失败，您可以选择手动执行 SQL 语句进行建表。

  * **操作步骤：** 请查阅项目文档 **`backend/docs`** 目录中的**启动流程或初始化文档**，找到相应的 SQL 脚本并手动输入到数据库中。
